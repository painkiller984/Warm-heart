Warm Heart is a storefront for soft plaids with calm tones and tactile textures. The project includes homepage, catalog, product card, cart, contacts, and auxiliary pages (about, FAQ, privacy).

### Stack

-   Vite for bundling and dev server
-   SCSS with component-level partials
-   Swiper for sliders
-   JustValidate for form validation

### Getting started

1. Install dependencies: `npm install`
2. Start dev server: `npx vite` (or `npm run dev` if you add a script)
3. Open the printed localhost URL in the browser

### Useful paths

-   Styles entry: `src/sass/style.scss`
-   Component styles: `src/sass/blocks/`
-   Scripts: `src/js/script.js`
-   Pages: `index.html`, `shop.html`, `card.html`, `cart.html`, `contacts.html`, `about.html`, `FAQ.html`, `privacy.html`

### Build (optional)

Vite config is in `vite.config.js`. Add a build script to `package.json` like `"build": "vite build"` if you plan to deploy.
